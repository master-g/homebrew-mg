# mg's Brew Tap

internal use only

## Installation

Get your Homebrew installed first, check [here](https://brew.sh/)

Add my Tap:

```sh
brew tap master-g/homebrew-mg
```

Install a Cask:

```sh
brew cask install i18n
```

## Updating

```sh
brew update
```

then

```sh
brew cask upgrade i18n
```
