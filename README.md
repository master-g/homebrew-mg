# mg's Brew Tap

internal use only

## Installation

Get your Homebrew installed first, check [here](https://brew.sh/)

Add my Tap:

```sh
brew tap master-g/tap
```

Install a Cask:

```sh
brew install master-g/tap/i18n
```

## Updating

```sh
brew update
```

then

```sh
brew upgrade master-g/tap/i18n
```
